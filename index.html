Exemplos de tags

Para marcar uma parte do texto usa a tag <mark>.

Para deixar uma parte do texto menor, usa a tag <small>.

Para deixar o texto em italico, usa a tag <i>.

Para deixar uma parte do texto em Negrito, usa a tag <strong>.

Para um texto sublinhado, usa a tag <u>.
